1.Клонировать репозиторий.
2.Скомпилировать проект с помощью Makefile
  make
3. Запустить программу
./dd_pow args
Аргументы для x^y pow(x, y) могут быть вида:
Два значения, head x и y: -xh x.head -yh y.head или 4 значения, head x tail x, head y tail y: -xh x.head -xt x.tail -yh y.head -yt y.tail
x - положительная
